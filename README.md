# AppointmentNote (โปรแกรมบันทึกการนัดหมาย)

โปรแกรม Windows Forms Application สำหรับบันทึกข้อมูลการนัดหมาย ช่วยให้ผู้ใช้สามารถจัดการตารางงาน ใส่หัวข้อ ระดับความสำคัญ และสถานะของงานได้อย่างง่ายดาย

เทคโนโลยีที่ใช้
* **Language:** C#
* **Framework:** .NET (Windows Forms Application)
* **IDE:** Visual Studio 2022 (หรือเวอร์ชันที่รองรับ)
วิธีดาวน์โหลดและติดตั้ง
1.  ไปที่ปุ่ม **Code** สีเขียวด้านบนของ Repository นี้
2.  เลือก **Download ZIP** หรือใช้คำสั่ง Git:
    ```bash
    git clone [https://github.com/YourUsername/AppointmentNote.git](https://github.com/YourUsername/AppointmentNote.git)
    ```
3.  แตกไฟล์ ZIP (หากดาวน์โหลดแบบไฟล์)

วิธีเปิดและรันโปรแกรม
1.  เปิดโปรแกรม **Visual Studio**
2.  เลือกเมนู **File > Open > Project/Solution**
3.  เข้าไปในโฟลเดอร์ที่ดาวน์โหลดมา เลือกไฟล์นามสกุล `.sln` (Solution File)
4.  กดปุ่ม **Start** (สีเขียว) หรือกดปุ่ม `F5` เพื่อรันโปรแกรม

คู่มือการใช้งาน (User Interface)

หน้าจอหลักประกอบด้วยส่วนรับข้อมูลดังนี้:

* **AppointmentID:** ช่องสำหรับกรอกรหัสการนัดหมาย (ห้ามเว้นว่าง)
* **Subject:** ช่องสำหรับกรอกหัวข้อการนัดหมาย (ห้ามเว้นว่าง)
* **Importance:** เลือกความสำคัญของนัดหมาย (Low, Medium, High)
* **Status:** เลือกสถานะของงาน (Planned, Confirmed, Completed)

ปุ่มควบคุม
* **Add:** กดเพื่อตรวจสอบความถูกต้องและบันทึกข้อมูล (แสดงผลทาง Message Box)
* **Clear:** ล้างข้อมูลทั้งหมดในหน้าจอให้เป็นค่าเริ่มต้น
* **Exit:** ปิดโปรแกรม

ตัวอย่างการใช้งาน

1.  กรอก **AppointmentID**: `AP-001`
2.  กรอก **Subject**: `ประชุมทีมโปรเจกต์`
3.  เลือก **Importance**: `High`
4.  เลือก **Status**: `Confirmed`
5.  กดปุ่ม **Add**
    * *ผลลัพธ์:* โปรแกรมจะแสดง MessageBox สรุปข้อมูลว่า "บันทึกข้อมูลสำเร็จ! รหัสการนัดหมาย: AP-001 หัวข้อ: ประชุมทีมโปรเจกต์..."

*หมายเหตุ: หากกรอกข้อมูลไม่ครบ (ช่อง ID หรือ Subject ว่าง) โปรแกรมจะแสดงข้อความเตือนและไม่ทำการบันทึก*
